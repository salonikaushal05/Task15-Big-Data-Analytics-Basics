#  Task 15 - Big Data Analytics Basics Using PySpark

##  Project Overview

This project was completed as part of the **Data Analytics Internship** at **Edutech Solution**.

The objective of this project is to understand the fundamentals of **Big Data Analytics** using **Apache Spark (PySpark)**. The project demonstrates how large-scale datasets can be processed efficiently through distributed computing and introduces core Big Data concepts such as the **5 Vs of Big Data**, **MapReduce**, and **Spark DataFrames**.

The analysis is performed on a client-hostname dataset using PySpark to showcase basic data loading, preprocessing, exploration, and analytical operations.


##  Objectives

* Understand the fundamentals of Big Data.
* Learn the architecture and working of Apache Spark.
* Perform distributed data processing using PySpark.
* Analyze a large dataset using Spark DataFrames.
* Explore data through filtering, grouping, sorting, and aggregation.
* Understand the concept of MapReduce and the characteristics of Big Data.


##  Dataset

The project uses a **Client Hostname Dataset** containing network-related information such as client IP addresses, hostnames, aliases, and address details. The dataset is analyzed using PySpark to demonstrate efficient processing of structured data in a distributed environment.
